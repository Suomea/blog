Spring Boot 文件上传
依赖：
```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-web</artifactId>
</dependency>
```

配置：
```yml
spring:
  servlet:
    multipart:
      enabled: true                   # 是否启用multipart上传，默认true
      max-file-size: 1MB              # 单个文件最大大小，默认1MB
      max-request-size: 10MB          # 整个请求最大大小，默认10MB
      file-size-threshold: 0B         # 文件大小阈值，超过此值会写入磁盘，默认0B
      location: ""                    # 临时文件存储位置，默认系统临时目录
      resolve-lazily: false           # 是否延迟解析，默认false
```
